gaussfit &gfit

gaussfit is an extremely flexible but rather temperamental general purpose fitting program written by William Jefferys et al. at the University of Texas. SDE provides a task, gfit, which is intended to insulate the user somewhat from the vagaries and complexities of gaussfit. It provides a number of prewritten gaussfit models which are a direct translation of the standard SDE variety, reads and writes gaussfit parameter files, runs the fitting program in a subshell and provides translation functions to interpret the results in the SDE environment. In spite of the intentions, you will need to understand the basics of gaussfit to use gfit effectively. See the manual in tools/gaussfit/man. A particularly nice feature of gfit is the ability to specify initial guesses to the program via SAOimage cursor files.
