Error messages

The following CHARACTER*(*) strings are defined in stdinc.h for use in ERRREPOR. 

 
 ERRFATAL 		 'Fatal error'


 ERRBDARG 		 'Bad subroutine argument'


 ERRNOMEM 		 'No memory available'


 ERRNTFND 		 'Requested item not found'


 ERRLOGIC 		 'Program logic error'


 ERRNOSLT 		 'No slot available'


 ERRWRGTP 		 'Item is of wrong type'


 ERRTRUNC 		 'Result truncated'


 ERRBADID 		 'Bad identifier supplied'


 ERRDB 		 'Internal database error'


 ERRINPUT 		 'Input read error'


 ERROUTPT 		 'Output write error'


 ERROPEN 		 'File open error'


 ERRCLOSE 		 'File close error'
