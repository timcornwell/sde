System-related activities: SYS routines

Access to the system is minimal. The main functions of the SYS routines are time and date retrieval, command execution, and logical name translation.

  • SYSDATEC (DTSTRING): is a subroutine which returns the date in the CHARACTER*(*) string DTSTRING in the format '23/11/87'
  • SYSDATET (DTSTRING): is a subroutine which returns the date and time in DTSTRING. The format is 'Sun Jan 22 21:15:11 1989'.
  • SYSETIME (DTSTRING): is a subroutine which returns the elapsed time and CPU (if available) in DTSTRING
  • SYSSYSTM (COMMAND): is an INTEGER function which executes a command line COMMAND on the host system, and returns an error status.
  • SYSTRANS (NAME1, NAME2): is a subroutine which performs logical name translation of NAME1 to NAME2. It calls getenv to translate any part it can.
