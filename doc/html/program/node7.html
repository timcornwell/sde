The main subroutines

SDE has a core set of subroutines which are vital to any working task. These must work for any task to live and breathe. We now describe these in detail.

  • Database access: DAT routines
  • Error handling: ERR routines
  • Messages: MSG routines
  • User interface: USR routines
  • Text input and output: TXT routines
  • String handling: STR routines
  • System-related activities: SYS routines
