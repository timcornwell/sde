Installing SDE on a UNIX machine

We don't support SDE and will discourage you from taking it away. However, it you decide to do so, then you will need to know how to install SDE on your machine.

  1. SDE is available by anonymous ftp from zia.aoc.nrao.edu (192.43.204.7) in directory /pub/sde.
  2. Get tar tape or file and run tar to extract all files into some convenient place (probably this should be the desired final home).
  3. Edit var/sdeini.hostname or var/sdeini.csh.hostname to insert definition of environment variables for your host. Either setenv or export SDEROOT to point to the relevant place and then either . or source the relevant initialization file to set up the environment variables for the next steps.
  4. Edit var/comopt.hostname to include compiler options for your host. Use this to override the options set in var/comopt.$MACHINE.
  5. run the script sdeinstall. This script will create all the necessary directories and compile everything.
Remember that there are support for or no guarantees to the /sde/ software. You are on your own!
