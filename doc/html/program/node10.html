Messages: MSG routines

The message system is very simple.

  • MSGWELCO (TEXT): writes the introductory message TEXT to the user, and also the data and time of compilation.
  • MSGPUT (MSG, TYPE): writes a message MSG which is CHARACTER*(*) to the user with a code TYPE, CHARACTER*1, which denotes what type of message it is. Valid types are 'I' for informative (most common), 'W' for warning, 'E' for fatal error, and 'D' for a debugging message.
