Text input and output: TXT routines

All text i/o should go through the text handling system. This uses string handles for files: these are just simple names rather than logical units.

  • TXTOPEN (HANDLE, FILE, STATUS): Open an external file FILE using a status STATUS, CHARACTER*(*) and associate it with a handle. STATUS can be one of READ, WRITE or READWRITE.
  • TXTCLOSE (HANDLE): closes the i/o stream with handle HANDLE.
  • TXTREAD (HANDLE, LINE, NCHAR, EOF): reads a line from the stream and returns the number of characters read NCHAR, INTEGER and an end-of-file indicator EOF, LOGICAL. A blank line returns NCHAR = 0 and EOF = .FALSE..
  • TXTWRITE (HANDLE, LINE): writes the line to the stream.
